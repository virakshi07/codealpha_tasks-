# 📌 Project Overview

This project focuses on predicting whether a breast tumor is **Malignant** or **Benign** using machine learning techniques. Multiple classification algorithms were trained and evaluated on the **Breast Cancer Wisconsin Dataset** to identify the most effective model for accurate diagnosis.

The project demonstrates how machine learning can support early disease detection by analyzing medical measurements and providing highly accurate predictions.

---

# 🎯 Objective

Develop and compare multiple machine learning models to accurately predict breast cancer diagnosis based on patient medical data, helping demonstrate the role of AI in healthcare and medical decision support.

---

# 🩺 Dataset

**Breast Cancer Wisconsin Dataset (Scikit-learn Built-in)**

- 📊 Total Samples: **569**
- 📋 Features: **30 Numerical Features**
- 🎯 Target Classes:
  - Benign
  - Malignant
- 🔀 Classification Type: **Binary Classification**

---

# 🤖 Machine Learning Models

The following classification algorithms were trained and evaluated:

| Model | Accuracy | ROC-AUC |
|--------|---------:|---------:|
| Logistic Regression | ~96% | ~99% |

# ▶️ How to Run

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/Disease-Prediction-from-Medical-Data.git
```

### 2️⃣ Navigate to the Project

```bash
cd Disease-Prediction-from-Medical-Data
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the Project

```bash
python disease_prediction.py
```

or simply open the notebook in **Google Colab** and run all cells.

> **No dataset download is required** since the Breast Cancer Wisconsin dataset is available directly through Scikit-learn.
