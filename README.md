# 🚀 CodeAlpha Machine Learning Internship

Welcome to my **CodeAlpha Machine Learning Internship** repository! This repository showcases the machine learning projects I completed during the internship, covering **speech processing, computer vision, and healthcare prediction** using Python and popular ML/DL frameworks.

---

## 📖 About

Throughout this internship, I explored real-world machine learning applications by building and evaluating models across different domains. Each project demonstrates a unique use case of artificial intelligence and highlights practical skills in data preprocessing, model training, evaluation, and deployment-ready workflows.

---

## 📂 Internship Tasks

| Task | Project | Description |
|------|---------|-------------|
| 🎙 **Task 2** | Emotion Recognition from Speech | Recognize human emotions from speech audio using MFCC features and a Deep Neural Network trained on the RAVDESS dataset. |
| ✍️ **Task 3** | Handwritten Character Recognition | Classify handwritten digits (0–9) using a Convolutional Neural Network (CNN) trained on the MNIST dataset. |
| 🩺 **Task 4** | Disease Prediction from Medical Data | Predict breast cancer diagnosis using multiple machine learning algorithms including Logistic Regression, Random Forest, and XGBoost. |
